Git操作

当你安装好git以后需要进行一个配置

  • git config --global user.name “wzs” 配置用户名
  • git config --global user.email "xxxxxxxxx@qq.com" 配置邮箱
  • cat ~/.gitconfig 可以查看git的配置

创建本地仓库

  • git init 初始化一个git仓库 初始化之后会默认有一个master分支
  • git status 查看仓库的状态
  • git add 文件名 可以将文件提交到暂存区
  • git add -A 将当前文件都加到版本控制中 git add all无论在哪个目录执行都会提交相应文件。
  • git add .只能够提交当前目录或者它后代目录下相应文件。
  • git rm --cached 文件名 恢复到之前的状态
  • git commit -m “本次提交的信息”
  • git push 提交

链接远程仓库

  • git remote add origin https://github.com/yourName/repositoryname.git 后面这个是远程仓库的路径
  • git push origin 分支名 在远程仓库创建一个分支
  • git push origin 本地分支名:远程分支名 在远程创建一个分支名和本地名不一样，但是有联系的分支

在本地删除远程分支

  • git push origin :分支名

拷贝远程仓库

  • git clone 地址 克隆远程代码
  • git pull 更新本地代码
  • git pull origin master 更新远程代码的的分支

git分支

  • git branch 分支名 创建新的分支
  • git chackout -b 分支名 创建并跳转分支
  • git branch -d 分支名 删除分支
  • git branch -D 分支名 强制删除分支
  • 合并分支(在主分支上) git merge 分支名

-git log --oneline 查看git日志
